# Simple Password Vault

A lightweight Android application built with React Native and Expo to keep your credentials organized and secure. No more forgetting passwords or using the same one everywhere.

## Why this project?
I built this app to solve a personal need: having a clean, private, and straightforward way to manage accounts without the clutter of enterprise managers. It's focused on speed and ease of use for day-to-day logins.

## Features
* **Encrypted Local Storage:** Your data stays on your device.
* **Minimalist UI:** Fast navigation and clean interface for quick access.
* **Search & Filter:** Find any account instantly as you type.
* **Custom Categories:** Organize passwords by Work, Social, Personal, etc.

## Tech Stack
* **Framework:** React Native (Expo SDK)
* **Navigation:** Expo Router
* **Storage:** [AsyncStorage]
* **Styling:** [Flexbox]
